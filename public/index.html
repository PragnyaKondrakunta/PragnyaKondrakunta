PATENT


My invention entitled Device and Method for Deep Learning Aided Custom Gesture Recognition involved the implementation of a sensor driven wearable that collects and feeds raw sensor data to a deep learning algorithm which classifies the performed gesture. The deep learning model is hosted on a server which communicates using law latency MQTT protocols. After applying for a patent for this technology, we founded FoGR (Future of Gesture Recognition) Tech.
